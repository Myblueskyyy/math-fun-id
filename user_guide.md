# Petunjuk Penggunaan

## PETUNJUK PENGGUNAAN APLIKASI
**Media Pembelajaran Matematika: Aritmatika Sosial**

Aplikasi ini dirancang untuk membantu siswa memahami konsep jual beli, untung rugi, diskon, dan pajak secara interaktif melalui materi, latihan, evaluasi, dan game edukasi.

### 1. Halaman Awal (Beranda)
* Saat membuka aplikasi, pengguna akan melihat menu utama.
* Pilih menu yang tersedia: Pre-Test, Materi, Post-Test dan Game Edukasi.

### 2. Pre-Test
* Pre-test bertujuan untuk mengetahui kemampuan awal pengguna.
* Kerjakan semua soal dengan jujur tanpa bantuan.
* Masuk ke menu Pre-Test untuk memulai.
* Setelah selesai, kamu dapat melihat hasil dan pembahasannya.

### 3. Materi Pembelajaran
Pada bagian ini terdapat beberapa submateri:
* Jual, Beli, Untung dan Rugi
* Diskon
* Pajak

**Cara penggunaan:**
* Pilih submateri yang ingin dipelajari.
* Bacalah penjelasan yang tersedia.
* Perhatikan contoh soal dan penyelesaiannya.
* Selesaikan soal diskusinya.
* Gunakan tombol "Selanjutnya" untuk berpindah materi.

### 4. Post-Test
* Post-test dilakukan setelah mempelajari materi pada tiap pertemuannya.
* Bertujuan untuk mengetahui peningkatan pemahaman.
* Kerjakan secara mandiri tanpa bantuan.
* Bandingkan hasil dengan pre-test untuk melihat perkembangan.

### 5. Game Edukasi
* Game dirancang untuk memperkuat pemahaman konsep secara menyenangkan.
* Ikuti instruksi permainan yang tersedia.
